Komponen Utama Komputer

Catatan pembelajaran tentang hardware komputer

Central Processing Unit (CPU)

CPU adalah otak dari komputer yang bertanggung jawab untuk menjalankan instruksi program dan melakukan perhitungan. CPU menentukan seberapa cepat komputer dapat memproses data.

Fungsi Utama:
  • Menjalankan instruksi program
  • Melakukan perhitungan aritmatika
  • Mengontrol aliran data dalam sistem
  • Mengelola operasi logika

CPU Model

Random Access Memory (RAM)

RAM adalah memori sementara yang digunakan untuk menyimpan data dan program yang sedang aktif. Semakin besar kapasitas RAM, semakin banyak program yang dapat dijalankan bersamaan.

Karakteristik:
  • Penyimpanan sementara (volatile)
  • Akses data sangat cepat
  • Menentukan multitasking capability
  • Data hilang saat power off

RAM Module

Motherboard

Motherboard adalah papan sirkuit utama yang menghubungkan semua komponen komputer. Semua komponen hardware terpasang dan berkomunikasi melalui motherboard.

Komponen Penting:
  • Socket CPU untuk processor
  • Slot RAM untuk memori
  • Slot ekspansi (PCIe)
  • Konektor power dan data
  • Chipset untuk kontrol sistem

Motherboard

Storage (HDD/SSD)

Storage adalah tempat penyimpanan permanen untuk data dan program. HDD menggunakan piringan magnetik, sementara SSD menggunakan flash memory yang lebih cepat.

Jenis Storage:
  • HDD: Kapasitas besar, harga murah
  • SSD: Kecepatan tinggi, tahan guncangan
  • NVMe: SSD dengan interface tercepat
  • Hybrid: Kombinasi HDD dan SSD

Graphics Card (GPU)

GPU bertanggung jawab untuk memproses grafik dan visual. Untuk gaming dan desain grafis, dedicated graphics card memberikan performa yang jauh lebih baik dari integrated graphics.

Kegunaan:
  • Rendering grafik dan video
  • Gaming dengan kualitas tinggi
  • Desain grafis dan video editing
  • Komputasi parallel (GPGPU)
  • Machine learning dan AI

Graphics Card

Power Supply Unit (PSU)

PSU mengkonversi listrik AC dari wall outlet menjadi DC power dengan berbagai voltase yang dibutuhkan komponen komputer. Kualitas PSU sangat penting untuk stabilitas sistem.

Fungsi PSU:
  • Konversi AC ke DC power
  • Menyediakan multiple voltage rails
  • Proteksi over-voltage dan over-current
  • Efisiensi energi (80+ rating)

Komponen Pendukung Komputer

Cooling System

Cooling System adalah sistem pendingin pada komputer yang berfungsi menjaga suhu komponen utama seperti CPU dan GPU agar tetap stabil. Dengan sistem pendingin yang baik, performa komputer akan lebih optimal dan umur hardware bisa lebih panjang.

Fungsi Utama:
  • Menjaga suhu komponen agar tidak overheat
  • Memastikan kinerja sistem tetap stabil
  • Meningkatkan umur pemakaian hardware
  • Mendukung performa maksimal saat beban kerja tinggi
Fun Fact:

Beberapa PC gaming dan server menggunakan liquid cooling yang bekerja seperti radiator mobil! Cairan dialirkan untuk menyerap panas, lalu didinginkan kembali agar suhu tetap rendah.

Casing (Chassis)

Casing adalah tempat untuk merakit semua komponen komputer, menyediakan proteksi fisik, serta membantu sirkulasi udara (airflow). Selain itu, casing juga menjadi elemen estetika dengan desain modern seperti tempered glass dan RGB lighting.

Karakteristik:
  • Melindungi komponen dari debu dan benturan
  • Menyediakan airflow untuk menjaga suhu
  • Memiliki slot ekspansi & drive bay untuk HDD/SSD
  • Mendukung cable management agar lebih rapi
  • Estetika (RGB, tempered glass)
